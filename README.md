# Workspace

A web-based application built using the following technologies:

- `TypeScript` as programming languages,
- `AstroJs` as a web framework,
- `Docker` as a containerization and deployment platform,
- and `GitHub Actions` as a CI/CD automation tool.

## Getting Started

1. Rename `.env.sample` to `.env` and update it with the required configuration values.

1. To build or rebuild the Docker images, run the command: `docker compose build`.

1. To start all services, run the command: `docker compose up -d`.

1. To stop the containers, run the command: `docker compose down`.
